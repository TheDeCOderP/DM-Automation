## ğŸ”„ Data Lifecycle

### 1. Generation Phase
- User provides prompt
- Optional reference image for sizing
- Gemini AI generates image
- Uploaded to `ai-generated-images/`

### 2. Selection Phase
- Image converted to File object
- Preview shown to user
- User can regenerate or proceed

### 3. Submission Phase
- Form submitted with image file
- Re-uploaded to appropriate folder
- URL saved to database

### 4. Storage Phase
- Permanent Cloudinary storage
- Database record created
- Relations established

### 5. Retrieval Phase
- URL fetched from database
- Image served via Cloudinary CDN
- Optimized delivery to users
